# Admin Guide

## Getting started
1. Sign in via magic link (invite email).
2. Complete workspace onboarding (name, timezone, locale).
3. Add connector (Google OAuth):
   - Click **Add Connector**.
   - Follow OAuth prompts.
   - Verify status shows **Connected**.
4. Configure SMTP:
   - Enter host, port, username, password/API key.
   - Send test email.

## Managing users
- Invite teammates by email.
- Assign roles (Admin, Member, Viewer).
- Revoke access via **Security > Sessions**.

## Feature flags
- Toggle AI summaries, suggested replies.
- Changes apply within 60 seconds.

## Monitoring
- View sync status per mailbox (last sync, errors).
- Check alerts in **Operations** panel.

## Support
- Contact DevOps for infrastructure issues.
- Report bugs via internal ticketing (Jira).
