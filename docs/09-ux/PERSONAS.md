# Personas

## P1 — Ops Manager (Internal Pilot)
- Needs quick overview of shared mailboxes.
- Prioritizes security and auditability.
- Moderate technical literacy.

## P2 — Knowledge Worker
- Sends/receives high volume of email.
- Values AI summaries and triage.
- Uses mobile occasionally; primarily desktop.

## P3 — Admin / IT
- Manages connectors, OAuth credentials, feature toggles.
- Responsible for compliance and monitoring.
